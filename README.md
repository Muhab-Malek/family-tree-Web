# 🌳 مشروع شجرة العائلة (family-tree-Web)

تطبيق ويب تفاعلي وبسيط ومصمم لعرض واستكشاف شجرة العائلة بشكل منظم وجذاب عبر المتصفح.

## ✨ الميزات
* عرض مرئي واضح لأفراد العائلة.
* تصميم متجاوب وسريع متوافق مع شاشات الجوال والكمبيوتر.

## 🚀 طريقة التشغيل
كل ما عليك فعله هو تحميل الملف وتشغيل `index.html` مباشرة على أي متصفح ويب.
